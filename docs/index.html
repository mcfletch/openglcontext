OpenGLContext A Learning Environment for PyOpenGL

Introduction

OpenGLContext is a testing and learning environment for PyOpenGL. It provides a generic scenegraph engine (based on VRML97's node mode) as well as basic context creation and interaction support in multiple GUI environments.

Features

  • Works on Multiple GUI Environments with Keyboard, Mouse and GUI events
    • Pygame
    • GLUT
    • wxPython
    • Qt (using the OpenGLContext_qt project)
  • Provides Basic Interactivity
    • 3D Flying Mode
    • Selection rendering based on unique-color buffers
    • Timer mechanisms and simple event handling
  • Geometry Types
    • Points (and Point-sprites)
    • Lines
    • Polygons
    • Trimmed NURBs
    • Extruded Shapes (with GLE)
    • Spheres, Cylinders, Cones
  • Textures (with PIL/Pillow), Materials
  • Shader Scenegraph Nodes
  • Optimizations
    • Caching Infrastructure
    • Frustum culling
  • Node-path Calculations
    • CPU-based calculations for forward/backward transformation matrices
  • OBJ Loader
  • VRML97 Loader (with SimpleParse installed)
  • BSP (Quake(TM) 3 File Loader/Viewer, with the Twitch project installed)
See the documentation page for more details.
